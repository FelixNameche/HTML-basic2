Qu'est-ce qu'une balise paire ?

Les balises paire constitue la majorité des balises html. Ont les nommes "paire" car elle vont toujours par 2, l'une ouvre et l'autre ferme.

  • <a> qui permet de créer des liens.
  • <div> qui permet les mise en page (en complément du CSS).
  • <p> qui permet de créer des paragraphes.
  • <ul> qui permet de créer des listes non-odronée.
  • <body> qui permet de différencier la tête du corps d'une page html.

Qu'est-ce qu'une balise orpheline ?

Les balises orphelines sont des balises qui n'ont pas besoin de balise de fermeture.

  • <img> qui permet d'ajouter des images.
  • <br /> qui permet le retour à la ligne.
  • <embed> qui permet l'insertion d'une application multimédia (vidéo YouTube par exemple).

Qu'est-ce qu'un attribut ?

Un attribut permet d'ajouter divers paramètres supplémentaire à certaines balises. Par exemple, l'attribut "_blank" placé dans une balise a permet d'ouvrir le lien dans une nouvelle fenètre.

Quel est la différence entre les balises de type Inline et Block ?

La différence entre un élément de type inline et un élément de type Block est que les "block" peuvent contenir plusieurs balises différentes alors que les éléments "inline" ne peuvent contenir qu'une seule balise. A noter que depuis la venue du HTML5 ces notions ont disparue.

    Exemples de balises Inline

  • <span>
  • <strong>
  • <img<>

    Exemples de balises Block

  • <div>
  • <h1>, <h2>, ..., <h6>
  • <table>
